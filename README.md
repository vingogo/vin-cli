# Vin CLI

Vin CLI 是一个开发 uni-app 应用的命令行工具。

## 安装

```shell
# 通过 npm
npm i @vingogo/vin-cli -g

# 通过 yarn
yarn global add @vingogo/vin-cli

# 通过 pnpm
pnpm add @vingogo/vin-cli -g
```

## 使用

- 使用帮助：

```shell
vin -h
```

- 创建一个开发模板：

```shell
vin template
```

- 启动 uniapp cli 项目开发模式：

```shell
vin dev
```

- 启动 uniapp cli 项目构建模式：

```shell
vin build
```
